## ✅ Características Implementadas

### 🚀 Rendimiento & SEO
- **Optimización LCP:** Precarga de fuentes e imágenes críticas.
- **Scroll Fluido:** Integración de **Lenis** para una navegación con inercia (momentum-based).
- **Lazy Loading:** Imágenes con carga diferida y decodificación asíncrona.
- **Content Visibility:** Uso de `content-visibility: auto` para mejorar el renderizado.

### 🎨 Diseño & Animaciones
- Modo **"Dark Luxury"** con efectos de glassmorphism.
- **ScrollSpy:** El menú se actualiza automáticamente según la sección visible.
- **Flip Cards 3D:** Interactividad avanzada en la sección de platillos.
- **Partículas Dinámicas:** Fondo animado con partículas flotantes.
- **Glow Effects:** Resplandores dorados sutiles en elementos clave.

### 📱 Responsive
- Totalmente adaptado para Desktop, Tablet y Mobile.
- Menú hamburguesa moderno con overlay animado.
